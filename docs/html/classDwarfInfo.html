Detailed Description

The DwarfInfo class provides functionality to retrieve information from DWARF debugging information.

The class encapsulates a DWARF_Debug object and provides methods to initialize the DWARF_Debug object, retrieve function information by RIP (Instruction Pointer), retrieve local variables of a function, and traverse the DWARF tree.
